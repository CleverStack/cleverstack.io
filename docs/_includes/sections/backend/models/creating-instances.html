Creating non-persistent instances
Creating a new model without saving it to the database
{% highlight javascript %} var user = new UserModel({ firstName : 'Richard', lastName : 'Gustin' }); {% endhighlight %}

And later, if you want to save the instance (persist) to the database:

{% highlight javascript %} // To persist the instance to the database (the same result as calling create like below) user.save().then(function(user) { /* { id: 1, firstName: 'Richard', lastName: 'Gustin' } */ }); {% endhighlight %}
Creating persistent instances
Creating a new model and persist it to the database
{% highlight javascript %} UserModel.create({ firstName : 'Richard', lastName : 'Gustin' }).then(function(user) { /* { id: 1, firstName: 'Richard', lastName: 'Gustin' } */ }) {% endhighlight %}
