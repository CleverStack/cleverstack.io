There are two different ways of defining a Field for a Model, firstly, to help make field definition quick and easy to define, or for when you only need to define the type of the Field, You can use the Field Type, for example String.

{% highlight javascript %} Model.extend({ // ... firstName: String, // ... }); {% endhighlight %}

Additionally, you can set the value to an Object containing the Option's for the Field, this is useful for when you need to supply other options, like required

{% highlight javascript %} Model.extend({ // ... firstName: { type: String, required: true }, // ... }); {% endhighlight %}
