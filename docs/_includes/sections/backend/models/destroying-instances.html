There are a couple of ways to delete instances stored in your database, lets take a look:

{% highlight javascript %} // 1) Destroying/deleting an instance directly user.destroy().then(function() { // DELETED }); // 2) Using Model.destroy() UserModel.destroy({firstName: 'Richo'}, {id: 1}).then(function(user) { // DELETED }); {% endhighlight %}

SQL Generated by the both ways in the above code:

{% highlight sql %} DELETE FROM `Users` WHERE `id` = 1; {% endhighlight %}
