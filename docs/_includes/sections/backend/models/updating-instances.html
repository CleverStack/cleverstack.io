There are a few ways to update the data stored in your database, lets take a look:

{% highlight javascript %} // 1) Updating an instance directly user.firstName = 'Richo'; user.save().then(function(user) { /* { id: 1, firstName: 'Richo', lastName: 'Gustin' } */ }); // 2) Using Model.update() UserModel.update({firstName: 'Richo'}, {id: 1}).then(function(user) { /* { id: 1, firstName: 'Richo' } */ }); {% endhighlight %}

SQL Generated by the both ways in the above code:

{% highlight sql %} UPDATE `Users` SET `firstName` = 'Richo' WHERE `id` = 1; {% endhighlight %}

Data Returned from Model.update()

It's important to note that Model.update() does not perform any SELECT's, meaning that after your UPDATE has completed you will only get back model field's that you updated or used as WHERE criteria.

If you want to have all of the records data returned (as apposed to Model.update), simply use the findAndUpdate method.

{% highlight javascript %} UserModel.findAndUpdate({firstName: 'Richo'}, {id: 1}).then(function(user) { // user will only contain all the data, because this method first // performs a SELECT before updating the record. }); {% endhighlight %}

SQL Generated by Model.findAndUpdate()

{% highlight sql %} SELECT * FROM `Users` WHERE `id` = 1; UPDATE `Users` SET `firstName` = 'Richo' WHERE `id` = 1; {% endhighlight %}
