Modules
Robust & Flexible Modules that are completely independent of other modules in the application.

CleverStack introduces a new and powerful approach to modules, that was designed with three goals in mind; flexibility, simplicity, and re-usability - as best practice, all functionality except what is in the CleverStack Core should be defined inside of modules.

CleverStack removes the complexity of installing & configuring modules and makes developing modules easier (vertical scale), and allows you to be able to re-use modules with little or no code change between projects. (horizontal scale, using the CLI).
