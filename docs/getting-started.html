Install CleverStack

Setup is made easy using our CleverStack Command Line Interface (CLI).

{% highlight bash %}$ npm install -g cleverstack-cli{% endhighlight %}

Important: Linux or Mac users may need to run this command using "sudo"

{% highlight bash %}$ sudo npm install -g cleverstack-cli{% endhighlight %}

Create a new project

To create a new project using the CleverStack CLI.

{% highlight bash %}$ clever init my-app{% endhighlight %}

Note: This command may take awhile depending on your internet connection and computer specifications.

Start serving

To start your development servers using the CleverStack CLI.

{% highlight bash %}$ cd my-app{% endhighlight %} {% highlight bash %}$ clever serve{% endhighlight %}

You are now serving http requests, open your web browser:

{% highlight bash %}http://localhost:9000{% endhighlight %}
