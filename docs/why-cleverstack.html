Why CleverStack?

CleverStack provides you with a cutting edge development workflow for AngularJS, NodeJS and other JavaScript driven web apps.


CleverStack CLI

  • Quick & Easy Install - Start new projects and install new features directly from your command line. The CleverStack CLI takes care of installation and configuration of all your project dependencies.

CleverStack Front-end

  • AngularJS Seed - We have a fully featured AngularJS seed. You can however use any front-end you wish with CleverStack.
  • AngularJS Modularity - Features are split into succinct AMD specification modules so they be added or removed from your project as you please.
  • AngularJS & RequireJS - Our Angular Seed is fully integrated with RequireJS to provide excellent front-end JavaScript dependency management.
  • Automated Unit Testing - Run automated unit tests using PhantomJS and produce code coverage reports.
  • Automated E2E Testing - Run automated end-to-end tests using Angular's Protractor.

CleverStack Back-end

  • NodeJS Seed - We have a fully featured NodeJS seed. We recommend you use this for the back-end.
  • NodeJS Modularity - NodeJS features are split into succinct AMD specification modules so they be added or removed from your project as you please.
  • NodeJS Background Tasks - Perform CPU intensive activites such as document uploading, PDF creation & virus scanning or run anything really in the background while keeping your app active.
  • Database ORM - Create your database tables and populate them with seed data specified in JSON.
  • Clever Controller - Create your own custom actions to request routes dynamically.
  • Automated Unit Testing - Run automated unit tests using PhantomJS and produce code coverage reports.
  • Automated E2E Testing - Run automated end-to-end tests using Supertest.

How does CleverStack work?

CleverStack uses technologies such as Node/NPM, Grunt & Bower and others to manage packages/dependencies and setup/manage your development workflow.

Node/NPM

CleverStack uses Node Package Manager (NPM) to install project package dependencies specified in package.json.

Grunt

CleverStack uses Grunt to manage JavaScript tasks such as providing you with a development server with live reload, running of tests or producing a production build of your app.

Bower

CleverStack uses Bower to manage front-end JavaScript dependencies and install them as components for your project.

Modularity

CleverStack promotes modularity where each module contains everything it needs to function including it's own package.json and it's own unit tests.
